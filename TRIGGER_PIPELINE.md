"# 🚀 GitLab CI/CD Pipeline Trigger Guide\n\nQuick guide to trigger E2E tests on GitLab CI/CD.\n\n## 📋 Table of Contents\n\n- [Quick Start](#quick-start)\n- [Methods to Trigger](#methods-to-trigger)\n- [Available Test Suites](#available-test-suites)\n- [Examples](#examples)\n- [Documentation](#documentation)\n\n---\n\n## ⚡ Quick Start\n\n### 1️⃣ Via GitLab UI (Easiest)\n\n1. Go to your GitLab project\n2. Navigate to **CI/CD** → **Pipelines**\n3. Click **\"Run pipeline\"** button\n4. Select branch and click **\"Run pipeline\"**\n\n### 2️⃣ Via Command Line\n\n```bash\n# Using Node.js\nnode scripts/trigger-gitlab-pipeline.js --env stg --suite regression\n\n# Using Python\npython scripts/trigger-gitlab-pipeline.py --env stg --suite regression\n\n# Using PowerShell\n.\\scripts\\trigger-gitlab-pipeline.ps1 -Env stg -Suite regression\n```\n\n---\n\n## 🎯 Methods to Trigger\n\n| Method | Difficulty | Use Case |\n|--------|------------|----------|\n| **GitLab UI** | ⭐ Easy | One-time triggers, manual testing |\n| **Command Line** | ⭐⭐ Medium | Automation, scripts, CI/CD integration |\n| **GitLab API** | ⭐⭐⭐ Advanced | Custom integrations, webhooks |\n| **Scheduled** | ⭐⭐ Medium | Regular automated testing |\n\n---\n\n## 🧪 Available Test Suites\n\n| Suite | Environment | Duration | Description |\n|-------|-------------|----------|-------------|\n| `regression` | STG | ~30 min | Full regression tests |\n| `api` | PROD (VN+JP) | ~15 min | API inference tests |\n| `smoke` | STG | ~5 min | Quick smoke tests |\n\n---\n\n## 💡 Examples\n\n### Run Regression Tests on Staging\n\n```bash\nnode scripts/trigger-gitlab-pipeline.js --env stg --suite regression\n```\n\n### Run API Tests on Production\n\n```bash\nnode scripts/trigger-gitlab-pipeline.js --env prod --suite api\n```\n\n### Run Tests on Specific Branch\n\n```bash\nnode scripts/trigger-gitlab-pipeline.js --branch develop --env stg --suite regression\n```\n\n### Wait for Pipeline to Complete\n\n```bash\npython scripts/trigger-gitlab-pipeline.py --env prod --suite regression --wait\n```\n\n### Dry Run (Preview)\n\n```bash\nnode scripts/trigger-gitlab-pipeline.js --env prod --dry-run\n```\n\n---\n\n## 📚 Documentation\n\n- **[Full Guide](docs/GITLAB_CI_CD_TRIGGER_GUIDE.md)** - Comprehensive documentation with all methods\n- **[Scripts README](scripts/README.md)** - Detailed script usage and examples\n\n---\n\n## 🔧 Setup\n\n### 1. Create `.env` file\n\n```bash\ncp scripts/gitlab-config.example .env\n```\n\n### 2. Configure GitLab credentials\n\nEdit `.env` file:\n\n```env\nGITLAB_URL=https://gitlab.com\nGITLAB_PROJECT_ID=your-project-id-or-path\nGITLAB_PRIVATE_TOKEN=glpat-xxxxxxxxxxxxxxxxxxxx\n```\n\n### 3. Install dependencies\n\n**Node.js:**\n```bash\nnpm install axios dotenv\n```\n\n**Python:**\n```bash\npip install python-dotenv requests\n```\n\n---\n\n## 🎉 Common Workflows\n\n### Daily Regression Tests\n\nSchedule to run every day at 2 AM:\n\n1. Go to **CI/CD** → **Schedules**\n2. Create new schedule with cron: `0 2 * * *`\n3. Set variables: `APP_ENV=stg`, `TEST_SUITE=regression`\n\n### Pre-deployment Tests\n\nRun full tests before deploying to production:\n\n```bash\nnode scripts/trigger-gitlab-pipeline.js --env prod --suite regression --wait\n```\n\n### Post-commit Tests\n\nAutomatically run smoke tests on every push:\n\nAdd to `.gitlab-ci.yml`:\n\n```yaml\nsmoke:\n  stage: regression\n  rules:\n    - if: $CI_PIPELINE_SOURCE == \"push\"\n  script:\n    - pnpm test:smoke\n```\n\n---\n\n## 📊 View Results\n\n### On GitLab\n- Go to **CI/CD** → **Pipelines**\- Click on pipeline to view details\n- Download artifacts (test reports)\n\n### HTML Reports\n\nDownload and open:\n```\nautomation-tests/packages/products/fpt-marketplace/reports/playwright-report/index.html\n```\n\n---\n\n## ❓ Troubleshooting\n\n| Issue | Solution |\n|-------|----------|\n| Pipeline not triggering | Check `.gitlab-ci.yml` and runner availability |\n| 401 Unauthorized | Verify `GITLAB_PRIVATE_TOKEN` is valid |\n| 404 Not Found | Check `GITLAB_PROJECT_ID` is correct |\n| Tests failing | Check logs in GitLab CI/CD job output |\n\n---\n\n## 🔗 Useful Links\n\n- [GitLab CI/CD Documentation](https://docs.gitlab.com/ee/ci/)\n- [Playwright CI/CD Integration](https://playwright.dev/docs/ci)\n- [GitLab Pipeline Triggers](https://docs.gitlab.com/ee/ci/triggers/)\n\n---\n\n## 📞 Support\n\nFor issues or questions:\n1.
